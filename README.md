# Mouse-following-fish
This is a small interactive webpage where a 150x84 png of a fish smoothly and gracefully swims toward your mouse. 

# Getting started
If you are reading this than your computer can likely run this.

# Installing
Download this -> https://github.com/fish-gaming/Mouse-following-fish.git 

# Executing program
Open index.html with whatever web browser you have.

# Limitations
At extremely high speeds, stretching may distort the fish image noticeably.
If the browser window is resized while paused, initial repositioning may behave differently.
